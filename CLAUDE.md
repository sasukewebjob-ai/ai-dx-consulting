# AIコンサルLP プロジェクト概要

## 目的・ゴール
名刺裏の右上QRコードから飛ぶ、**法人・経営者向けの「AI・DXコンサル」LP**。
名刺交換した相手（経営者・企業担当者・見込み客）に「この人に相談すると何ができて、どんな成果があるか」を伝え、公式LINEの無料相談へ繋げる。

- 名刺表のQR ＝ ポートフォリオサイト
- 名刺裏のQR ＝ このLP

## ポジショニング
> AIを「教える」だけで終わらせない。業務を「速く」し、集客を「仕組み」にするところまで伴走する。

AIコンサル × SEO × Web × ツール開発（Claude Code）が一人で繋がっているのが差別化の核。
名刺表の肩書き「個人事業主・経営者のための AIによる業務改善が得意な AI・DXコンサル／Webマーケター」と一致。

## 技術スタック
- 単一HTML（`index.html`）で完結。ライブラリ・ビルド不要
- デザインは「LP作成」（ライトザライト）のデザインシステムを踏襲：
  生成り背景＋ゴールド/グリーン、明朝（Noto Serif JP）見出し、Notion×Stripe系
- `profile.jpg` … 根津さんプロフィール写真（LP作成から流用・480px圧縮済）

## セクション構成
1. Hero … 「個人事業主・経営者のための、AIによる業務改善。」＋実績3数字
2. Problem（黒背景）… 経営者の悩み6つ
3. Solution … 差別化コンセプト＋フロー（相談→業務設計→実装→定着）
4. Before/After＋数字 … 実装実績ベースの業務短縮試算（後述）
5. Services … 提供6メニュー（貢献→ベネフィット付き）
6. Track Record … ポートフォリオ30件への誘導
7. Profile … 経歴・実績・メディア掲載
8. Flow … 相談の流れ
9. FAQ … アコーディオン
10. Final CTA ＋ スマホ追従ボタン

## Before/After 数字（実装実績ベースの試算）
誇張せず、定型度が高い業務ほど削減率を大きくしている。客先の声は捏造せず「試算」として明示。
| 業務 | Before→After | 根拠 |
|---|---|---|
| SEO記事・ブログ | 8h→2.5h（1/3） | SEOブログ自動執筆 |
| 提案・社内資料 | 6h→1.5h（1/4） | スライド自動生成 |
| 見積・定型帳票 | 30分→3分/件（1/10） | 見積書・帳票生成 |
| 申込・問い合わせ | 15分/件→ほぼ0 | Web申込システム |
| SNS動画リール | 2h→20分/本（1/6） | リール自動生成 |
| 進捗管理・催促 | 週5h→週1h（80%減） | LINEタスク管理 |

## モーション（2026-07-09導入・LP作成から移植）
- `.fade`（下から40px）＋`.fade-l`／`.fade-r`（左右56px）／`.fade-zoom`＋親`.stagger`で時間差登場。Heroは`heroIn` keyframesで順次登場
- 表示1.8s後にJSがfade系クラスを除去（カード本来のhoverに復帰）。`prefers-reduced-motion`対応済み
- **注意**：`.ba-arr`・`.ladder-arr`はモバイルで`rotate(90deg)`を持つためfadeを付けないこと
- 実装前バックアップ：`index-pre-motion.html`／詳細な技術メモは`LP作成/作業ログ.md`の2026-07-09参照
